QUICK REFERENCE GUIDE FOR JEN-CHIEH'S .EMACS KEYBINDINGS
-----------------------------------------------------------
=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=
* Some Notes:                   *
*     C-S : Control + Shift     *
*     C-M : Control + Alt       *
=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=

ESC             Exit minibuffer and go to the top level.

TAB             Cycles through completions for a word.

M-B             Switch to buffer in the current window.
M-S-B           Switch to buffer in the other window.

M-F             Load file in the current window.
M-S-F           Load file in the other window.

M-s             Save the file in the current window.
M-m             Looks for and executes a build.bat.

M-w             Switch to the other window. (Next)
M-q             Switch to the other window. (Prev)
M-e             Aced window. (more advanced of the switch window functionality)

M-K             Kills a buffer (ask to save it first if it's modified).

C-arrows        Move cursor by larger increments (word, whitespace)

C-SPACE         Sets the mark.
C-TAB           Reindent block.
C-q             Copies the block.
C-f             Pastes the block.
C-w             Cuts the block.
M-S-;           Pops back to the previous mark.

M-u             Undo

M-j             Jump to a function by name.

M-.             Reflow a comment.

M-b             Go to the next error after a compile.
M-S-n           Go to the previous error after a compile.

M-g             Goto line number.

C-c d           duplicate the line to the next line.
C-d             Kill the whole line.
C-c ;           Rename the whole text with the same name.
C-r C-r         Rename the whole text with the same name.

C-x h           mark whole buffer (Select All)
C-a             mark whole buffer (Select All)

M-]             Run the program (run.linux/run.bat/run.mac)
C-<f5>          Run the program (run.linux/run.bat/run.mac)

C-S-b           Build the program (build.linux/build.bat/build.mac)

C-u anykey      insert 4 of that key
M-i             insert 4 spaces
C-m             jump to next line and indent if needed (depend on next line)

M-k             Maybe kill the buffer? Check if the file in open
                in other window. If there are buffer still exist,
                do not kill the buffer. Just switch to the previous
                buffer in the current frame. Otherwise just kill
                the buffer.
M-K             Kill the current file, not maybe. (kill-this-buffer)

C-c C-c         copy
C-x C-x         cut
C-v             paste
C-z             undo (undo-tree-undo)
C-y             redo (undo-tree-redo)

C-f             Search forward / Search pattern (Start increament search)
                  => C-s  Moves to next occurrence.
                  => C-r  Moves to previous occurrence.
C-x C-f         Search pattern in project root.

C-rr            Rename the whole text with the same name.
C-rp            Replace pattern in project root.
C-xC-s          Save the all the buffer that changes in the 
                `ag' replace Menu.            

C-s             Untabify the document then save the file.
C-S             Tabify the document then save the file.

C-/             comment-region
C-?             uncomment-region

C-p             package-list-packages
M-=             text-scale-increase
M--             text-scale-decrease

C-k C-f         format select region
C-k C-d         format current document
C-S-f           format the region if region select else format the document.

M-U             uppercase-word, if region do region
M-D             downcase-word, if region do region

C-x-a           align all variables in the region if region selected, 
                else align full document insteead.

M-m             open buffer menu.
M-M             open buffer menu in other window.

M-r             reload the file withou asking.

C-r e           rename current file.

M-right         Jump to next capital letter.
M-left          Jump to previous capital letter.

M-backspace     Kill until next capital letter appears, backward.
M-S-backspace   Kill util next capital letter appears, forward.

C-r backspace   Delete the same character at current cursor's 
                position backward repeatedly util it meet different 
                character.
C-r S-backspace Delete the same character at current cursor's 
                position forward repeatedly util it meet different 
                character.

C-x C-v         Reload '.emacs' file in a proper way.

C-w o           Start httpd server for real time editing markdown
                using `impatient-mode'.
C-w p           Close httpd server for real time editing markdown
                using `impatient-mode'.

C-h h           Toggle window split between horizontal and 
                vertcial.

C-w e           Toggle enlarge current selected window.
				
M-p             Open list of processes. (`list-processes')

M-up            Move current line up.
M-down          Move current line down.

C-M-up          Scroll window up one line without moving cursor.
C-M-down        Scroll window down one line without moving cursor.

C-c f           Change font style.

C-return        Binds to `jcs-ctrl-return-key'. If follows the list 
                of function error handling. If something wrong with 
                the order down function, the next function will be 
                call to replace the previous function. The priority 
                function list is list below: 
                  (1) goto-address-at-point
                  (2) custom-abbrev-complete-word

C-r o           Read-Only mode toggle.

C-r f           Open recent files.

C-t             Toggle `tabbar-mode'.

                  -> package-menu-mark-delete

                  -> package-autoremove

=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=
* Cross Language Mode
=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=
f12             goto declaration/definition on same window.
S-f12           goto declaration/definition on other window.

f7              open the same file on other window.
f8              If there is corresponding file then open that. Otherwise 
                the functionality is the same as f7 above.

C-S-B           Run the build script in the project. If cannot find 
                the file in the project it will find the file in just 
                the current directory instead. (.bat, .linux, .macosx, etc.)
f5              Run the run script in the project. If cannot find 
                the file in the project it will find the file in just 
                the current directory instead. (.bat, .linux, .macosx, etc.)

C-x t           Open todo list file in the project. If cannot find 
                the file in the project it will find the file in just 
                the current directory instead. [Default: other window]
C-x u           Open update log file in the project. If cannot find 
                the file in the project it will find the file in just 
                the current directory instead. [Default: other window]

=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=
* Navigating General Programming Symbols
=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=
M-)             Search forward next close parenthesis.
M-(             Search backward next open parenthesis.
M-]             Search forward next close square parenthesis.
M-[             Search backward next open square parenthesis.
M-}             Search forward next close curly parenthesis.
M-{             Search backward next open curly parenthesis.

M-'             Search forward next '. (single quotation mark)
M-;             Search backward next '. (single quotation mark)

M-"             Search forward next ". (double quotation mark)
M-:             Search backward next ". (double quotation mark)

M->             Search forward next >. (Greater than sign)
M-<             Search backward next <. (Less than sign)

M-S-<right>     Search forward at current point.
M-S-<left>      Search backward at current point.

M-.             Search forward next ,. (comma)
M-,             Search backward next ,. (comma)
C-M-.           Search forward next .. (period)
C-M-,           Search backward next .. (period)

-- Org Mode --
S-<up>          Move cursor up one row table.
S-<down>        Move cursor down one row table.
S-<right>       Move cursor right one column table.
S-<left>        Move cursor left one row column table.
